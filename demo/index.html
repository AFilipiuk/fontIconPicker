jQuery fontIconPicker v1.0.0

jQuery fontIconPicker is a small (1.58kb gzipped) jQuery plugin which allows you to include a simple icon picker with search and pagination inside your administration forms. The list of icons can be loaded manually using a SELECT field, loading an array of icons or directly from a Fontello config.json file.

In firefox fontIconPicker icons won't be shown correctly because of CORS. For the same reason "Load icons from Fontello JSON config file" won't work on Chrome or IE 10. If you need to do some local testing you can disable strict_origin_policy at your risk.

How it works

Just include a copy of jQuery, the fontIconPickers script, the fontIconPickers theme and your Font Icons... Now you can trigger it on a SELECT or TEXT input... check the first example to start!

<!-- jQuery -->
<script src="jquery-1.7.1.min.js"></script>

<!-- fontIconPicker -->
<script src="jquery.fonticonpicker.min.js"></script>
<link rel="stylesheet" href="grey-theme/jquery.fonticonpicker.min.css">

<!-- Font Icons -->
<link rel="stylesheet" href="fontello-7275ca86/css/fontello.css">

Here's fontIconPicker options:

var $picker = $('.picker').fontIconPicker({
    source:         false,      // Icons source (array|false)
    emptyIcon:      true,       // Empty icon should be shown?
    iconsPerPage:   20,         // Number of icons per page
    hasSearch:      true        // Is search enabled?
});

And it's only custom method .loadIcons() which lets you load icon sets on the fly.

$picker.loadIcons(['icon-one', 'icon-two']);

Basic: Load from SELECT input field

A SELECT input field is the best element to trigger fontIconPicker because it beautifully degrade when js fail. The input value will be used for the current selected icon.

HTML 
<select id="myselect" name="myselect">
    <option value="">No icon</option>
    <option>icon-user</option>
    <option>icon-search</option>
    <option>icon-right-dir</option>
</select>
JS 
$('#myselect').fontIconPicker();

Load from TEXT input field (no empty icon, no search)

You can of course trigger fontIconPicker on a TEXT field, loading the icon set through the source option. In this example we're going to disable search and the ability to leave fontIconPicker empty.

HTML 
<input type="text" name="mytext" id="mytext"></input>
JS 
$('#mytext').fontIconPicker({
    source:    ['icon-heart', 'icon-search', 'icon-user', 'icon-tag', 'icon-help'],
    emptyIcon: false,
    hasSearch: false
});

Load icons from Fontello JSON config file

Without having to manually populate fonts you can load them from a json file. Following an example using Fontello config.json file. fontIconPicker will be first instantiated with a loading icon, when ajax does its thing icons will magically appears! If something goes wrong, before asking for help, remember CORS!!

Load icons from Fontello config.json file HTML 
<input type="text" name="mytext" id="mytext" value="icon-heart"></input>
<a href="#" class="loadicons">Load icons from Fontello config.json file</a>
JS 
var icons = [], myIconPicker = $('#mytext').fontIconPicker();

$('loadicons').click(function (){
    $.ajax({
        url:         'config.json',
        dataType:    'JSON'
    }).done(function (r) {
        $.each(r.glyphs, function (i,v) {
            icons.push(r.css_prefix_text + v.css);
        });
        myIconPicker.setIcons(icons);
    });
});

fontIconPicker validation

Since fontIconPicker is nothing more than an interface to a common SELECT or TEXT input you can easily implement the validation on the original item.

HTML 
<select id="myselect" name="myselect">
    <option value="">No icon</option>
    <option>icon-user</option>
    <option>icon-search</option>
    <option>icon-right-dir</option>
</select>
JS 
$('#myselect').fontIconPicker();

$('#myform').submit(function () {
	
    $(this).find('span.error').remove();
	
    if (!$('#myselect').val()){
        $('#myselect').after('<span class="error">Required</span>');
        return false;
    }
    
    // Continue...

});

Change icon set on the fly

You can use .setIcons() also to load a custom set of icons on the fly. If the current selected icon is not there, the empty icon will be used ( or the first icon of the set if emptyIcon: false ).

Load: All | Icon set 1 | Icon set 2 HTML 
<input type="text" name="mytext" id="mytext" value="icon-heart"></input>
<a href="#" class="all">All</a>
<a href="#" class="filter1">Icon set 1</a>
<a href="#" class="filter2">Icon set 2</a>
JS 
var icons = ['icon-heart', 'icon-search', 'icon-user', 'icon-tag'],
    myIconPicker = $('#mytext').fontIconPicker({
        emptyIcon: false,
        source: icons,
        iconsPerPage: 30
    });

$('.all').click(function () {
    myIconPicker.setIcons(icons);
    return false;
});

$('.filter1').click(function () {
    myIconPicker.setIcons(['icon-heart', 'icon-search']);
    return false;
});

$('.filter2').click(function () {
    myIconPicker.setIcons(['icon-user', 'icon-tag']);
    return false;
});

Trigger event on icon change

Change event is triggered on the element on icon change so you'll be able to trigger custom action each time a icon get selected.

HTML 
<input type="text" name="mytext" id="mytext"></input>
JS 
$('#mytext').fontIconPicker({
    source: ['icon-heart', 'icon-lemon', 'icon-user', 'icon-tag']
}).change(function () {
    if ( $(this).val() ){
        $('.current_icon').html('Current icon is <i class="' + $(this).val() + '"></i>');
    } else {
        $('.current_icon').text('No icon selected');
    }
});

ACF { fontIconPicker WordPress plugin

I've first written jQuery fontIconPicker in order to build an Advanced Custom Field plugin to associate Font icons to WordPress categories. So here it is, ACF { fontIconPicker, the best real world example of jQuery fontIconPicker use.

Download

Install ACF { fontIconPicker

Just upload the 'acf-fonticonpicker' folder to your 'wp-content/plugins/' folder and activate the plugin... that's it! Alternatively you can include the plugin inside your theme: copy the 'acf-fonticonpicker' folder into your theme folder (can use sub folders) and edit your functions.php file adding the code below.

PHP 
include_once('acf-fonticonpicker/acf-fonticonpicker.php');

Use ACF { fontIconPicker

Once you have added and valorized fontIconPicker field you can output an icon with the following piece of code:

PHP 
if ( get_field('myicon') ){
    echo "<i class="'.get_field('myicon').'"></i>";
}

Load your own set of Font Icons

ACF { fontIconPicker provide an hook to set your own theme custom set of Font Icons. Just download a set of icons from Fontello (or build one using the same file and folder settings) and upload it in your theme folder. Now add this little piece of code to your functions.php file and you're done!

PHP 
/**
 * Filter ACF { fontIconPicker settings
 *
 * @param Array $settings Custom field settings array
 */
add_filter( 'acf/acf_field_fonticonpicker/settings', 'edit_fonticonpicker_settings' );
function edit_fonticonpicker_settings( $settings ) {

    // path to the icon set config.json file
    $settings['config'] = get_stylesheet_directory() . '/fontello-b553ae82/config.json';

    // uri to the icon set css file
    $settings['icons'] = get_stylesheet_directory_uri() . '/fontello-b553ae82/css/fontello.css';

    return $settings;

}
